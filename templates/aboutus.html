Anomaly-Based Network Intrusion Detection System Using Ensemble Machine Learning is a Machine Learning-powered solution designed to detect malicious network activities by identifying unusual traffic patterns. Using the CICIDS2017 Dataset, our system learns to classify both normal and attack behaviors, helping strengthen network defenses against cyber threats.

üéØ Project Objective

The main objective of this project is to develop an anomaly-based network intrusion detection system that can accurately identify malicious network activities by leveraging the power of ensemble machine learning techniques. The system aims to enhance the detection accuracy, minimize false positives, and adapt effectively to evolving cyber threats.


By integrating multiple machine learning algorithms such as Random Forest, Cat Boosting, Decision Tree, KNN, and Logistic Regression, the project seeks to combine the strengths of individual models to achieve a more robust and reliable intrusion detection framework.
