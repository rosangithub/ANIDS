About Our Project

Anomaly-Based Network Intrusion Detection System Using Ensemble Machine Learning is a Machine Learning-powered solution designed to detect malicious network activities by identifying unusual traffic patterns. Using the CICIDS2017 dataset, our system learns to classify both normal and attack behaviors, helping strengthen network defenses against cyber threats.

üéØ Project Objective

The main objective of this project is to develop an anomaly-based network intrusion detection system that can accurately identify malicious network activities by leveraging the power of ensemble machine learning techniques. The system aims to enhance the detection accuracy, minimize false positives, and adapt effectively to evolving cyber threats. By integrating multiple machine learning algorithms such as Random Forest, Cat Boosting, Decision Tree, KNN, and Logistic Regression, the project seeks to combine the strengths of individual models to achieve a more robust and reliable intrusion detection framework. Additionally, the project aims to: Analyze the CICIDS2017 dataset to extract meaningful features for distinguishing normal and attack traffic. Build and evaluate ensemble-based models capable of detecting various network intrusions in real time. Provide a user-friendly web dashboard to visualize detection results and performance metrics. Contribute to the field of cybersecurity by proposing a scalable and efficient approach to network threat detection using machine learning.
